# ddz-server

## 安装
```sh
git clone https://github.com/Deasoso/CryptoLandlords-server
cd ddz-server
composer install

# 启动服务
php server.php
```

WebSocket 服务器默认监听 `0.0.0.0:34567` ，本地客户端可使用地址 `ws://localhost:34567` 连接。

服务器监听地址可以通过编辑 `server.php` 修改。
